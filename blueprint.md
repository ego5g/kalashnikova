# Веб-сайт для персонального тренера Дарьи Калашниковой

## Обзор

Это одностраничный лендинг (landing page) для персонального фитнес-тренера Дарьи Калашниковой. Сайт представляет информацию о тренере, ее услугах, программах тренировок, результатах клиентов и предоставляет контактные данные.

## Дизайн и стили

- **Основная тема:** Темная тема с использованием черных и темно-серых фонов.
- **Акцентные цвета:** Градиенты от фиолетового (`purple-500`) до синего (`blue-500`).
- **Шрифты:** Стандартный шрифт `font-sans` из Tailwind CSS.
- **Визуальные эффекты:** Размытые фоновые круги для создания глубины, тени для "приподнятия" карточек, интерактивные эффекты при наведении.

## Структура и компоненты

Сайт состоит из следующих секций:

1.  **Header (Шапка):** Фиксированная шапка с навигацией по сайту.
2.  **Hero Section (Главный экран):** Заголовок, краткое описание и кнопка призыва к действию.
3.  **Gallery (Галерея):** Сетка с фотографиями.
4.  **About (Обо мне):** Информация о тренере.
5.  **Programs (Программы):** Карточки с описанием и стоимостью программ.
6.  **Reviews (Отзывы):** Отзывы клиентов.
7.  **Contact (Контакты):** Призыв к действию и кнопка для связи.
8.  **Footer (Подвал):** Копирайт.

## План текущих изменений (Улучшение адаптивности)

1.  **Создать мобильную навигацию:**
    *   Вынести компонент шапки в отдельный файл `app/components/Header.tsx`.
    *   Превратить его в клиентский компонент (`"use client"`) для использования состояния.
    *   Добавить иконку "гамбургера" для мобильных экранов.
    *   Реализовать полноэкранное меню, которое открывается и закрывается по клику на иконку.

2.  **Адаптировать размеры шрифтов:**
    *   Изменить размеры заголовков и текста в секциях `Hero`, `About`, `Programs`, `Reviews`, `Contact`, чтобы они были меньше на мобильных устройствах и увеличивались на десктопных. (например, `text-4xl md:text-5xl`).

3.  **Проверить и скорректировать отступы:**
    *   Убедиться, что вертикальные и горизонтальные отступы выглядят гармонично на всех размерах экранов.
