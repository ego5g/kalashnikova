# Веб-сайт для персонального тренера Дарьи Калашниковой

## Обзор

Это веб-сайт для персонального фитнес-тренера Дарьи Калашниковой. Сайт будет реструктурирован из одностраничного лендинга в многостраничное приложение для улучшения навигации, поддержки и исправления ошибок на мобильных устройствах.

## Дизайн и стили

- **Основная тема:** Темная тема с использованием черных и темно-серых фонов.
- **Акцентные цвета:** Градиенты от фиолетового (`purple-500`) до синего (`blue-500`).
- **Шрифты:** Стандартный шрифт `font-sans` из Tailwind CSS.
- **Визуальные эффекты:** Размытые фоновые круги для создания глубины, тени для "приподнятия" карточек, интерактивные эффекты при наведении.

## План реструктуризации

1.  **Создать директорию `components`:**
    *   `components/Header.tsx`: Компонент с навигацией и логотипом.
    *   `components/Footer.tsx`: Компонент с контактной информацией и ссылками.
    *   `components/ThemeSwitcher.tsx`: Переместить существующий `ThemeSwitcher` в эту директорию.

2.  **Создать страницы в директории `app`:**
    *   `app/programs/page.tsx`: Страница с описанием программ тренировок.
    *   `app/reviews/page.tsx`: Страница с отзывами клиентов.
    *   `app/about/page.tsx`: Страница с информацией о тренере.
    *   `app/contact/page.tsx`: Страница с контактной формой и информацией.

3.  **Обновить `app/layout.tsx`:**
    *   Импортировать и использовать `Header` и `Footer`.
    *   Обеспечить общую разметку для всех страниц.

4.  **Обновить `app/page.tsx` (Главная страница):**
    *   Оставить только ключевую информацию: приветствие, краткое описание и призыв к действию.
    *   Удалить секции, которые будут перенесены на отдельные страницы.

5.  **Исправить ошибки отображения на мобильных устройствах:**
    *   В `components/Header.tsx` обеспечить, чтобы мобильное меню имело сплошной фон и не вызывало смещения контента.
    *   Исправить прозрачность кнопки смены темы.
